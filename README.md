# Text Repeater
Generates repeating text to use as a watermark or to print. 

Can configure:
- Text
- Text Color
- Background Color
- Angle
- Vertical Spacing
- Word Spacing
- Word Size

There is a download button to download the generated image. 

Demo available at https://automontronic.github.io/TextRepeater/
